# Next.js Dashboard

A financial dashboard application built with Next.js App Router. The application allows users to manage invoices and customers, track revenue, and monitor pending payments through a clean, responsive interface.

Key features covered:
- Authentication with NextAuth.js
- Server-side rendering and data fetching with PostgreSQL
- Server Actions for form handling and mutations
- Incremental static regeneration and route-level streaming
- Form validation with Zod
- Accessible, responsive UI with Tailwind CSS

This project was completed as part of the official [Next.js App Router Course](https://nextjs.org/learn) provided by Vercel.
